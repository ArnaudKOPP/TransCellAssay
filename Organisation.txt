Screen Object Model

    Screen, Plate, Replicat, PlateSetup object

Statistics part
-Statistics
    --Normalization
        ---SystematicError
        ---VariabilityNormalization
    --QC
    --Score
        ---SingleCellScore
    --Test

-IO
    --DataBase
    --FileFormat


Graphics, WellFormat, ...
-Utils


Machine Learning part
-Machine Learning
    --Classification
    --Clustering
    --DimensionalityReduction
        ---supervised
        ---unsupervised


Bayesian stats part
-Bayesian


Dose Response Analysis part like IC50
-DoseResponseAnalysis
